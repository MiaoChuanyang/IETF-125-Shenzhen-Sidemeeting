# IETF-125-Shenzhen-HFC-Sidemeeting

We're planning to have a side meeting at IETF 123: Discussion on use case and requirements of Hybrid function chain(HFC). 

Backgroud information: In cloud-native environments and service mesh scenarios, service chains offer a highly flexible approach to application implementation. However, existing service-layer chaining mechanisms rely solely on L7 (application layer) requirements and lack coordination with L3 (network layer) quality, resulting in deficiencies in end-to-end service quality across the entire runtime. This side meeting is proposed to discuss the use cases, requirements, and any potential standardization work for a hybrid function/service chain that harmonizes service requirements with both network and computing resources.
